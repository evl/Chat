Collection of chat frame tweaks

h2. Configuration

* Enable and disable modules by modifying @evl_Chat.toc@
* Copy @config.lua.sample@ to @config.lua@ and modify as required.

h2. Features

* Hide chat frame buttons
* Hide chat box background and focus
* Modify edit-box anchor
* Automatically resolve in-game links to items/quests/achievements etc. for custom chat channels
* Repeat spam filter
* Copy chat contents (located under the right-click menu of any chat tab)
* Clickable URLs
