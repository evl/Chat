Collection of chat frame tweaks

h2. Configuration

* Enable and disable modules by modifying @evl_Chat.toc@
* Copy @config.lua.sample@ to @config.lua@ and modify as required.

h2. Features

* Mouse-wheel scrolling of chat windows
* Sticky channel input
* Hide buttons
* Clickable URL links in chat w/copy dialog
* Copy chat contents (located under the right-click menu of any chat tab)
